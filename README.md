# fastway-couriers-db
A relational database designed for a courier management system, built using Microsoft SQL Server (SSMS).

## How to Run
Execute the scripts in this order in SSMS:
1. `1_Creating_the_FastwayCourier_Database.sql` — creates all tables and indexes
2. `2_Entering_Sample_Data.sql` — inserts sample data
3. `3_All_Views.sql` — creates all views
4. `4_Procedures_Triggers_and_Login.sql` — adds stored procedures, triggers, and logins

## Built With

- T-SQL (Transact-SQL)
